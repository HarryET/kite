# Features

- **HTTP Interactions** <br>
  Uses a HTTP server for interactions meaning no gateway connections
- **HTTP Client** <br>
  Has a build in HTTP client to make requests easy

# Example

```dart
final kite = Kite(
    "publicKey",
    token: "aaa.bbb.ccc", // Bot's Token, can be null
  );
```
